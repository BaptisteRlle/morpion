# Jeu du Morpion

Un jeu de Morpion avec 4 modes de jeu différents pour varier les plaisirs !

## Comment démarrer

1. Ouvrez le fichier `morpion.html` dans votre navigateur
2. Entrez les noms des deux joueurs
3. Choisissez votre mode de jeu

## Les 4 modes de jeu

### Morpion Simple
Le jeu classique ! Alignez 3 symboles (X ou O) pour gagner.

### Morpion Ultimate
9 petites grilles forment une grande grille. Votre coup décide où l'adversaire doit jouer ensuite. Gagnez 3 petites grilles alignées pour gagner la partie !

**Exemple :** Si vous jouez dans la case en haut à droite d'une petite grille, votre adversaire devra jouer dans la grille en haut à droite de la grande grille.

### Morpion 2.0
Attention, vous ne pouvez avoir que 3 symboles sur la grille ! Votre 4ème coup remplace automatiquement votre plus ancien symbole. Un jeu de stratégie et de mémoire.

### Morpion VS Bot
Affrontez l'ordinateur ! Le bot choisit ses cases aléatoirement. Parfait pour s'entraîner.

## Boutons disponibles

- **Recommencer** : Nouvelle partie sans perdre les scores
- **Reset Score** : Remettre les scores à zéro
- **Retour Accueil** : Retour au menu pour changer de mode

## Technologies

HTML, CSS et JavaScript pur. Aucune installation nécessaire !
